JavaScript Basics

what's the difference between let, const, and var?

let can be reassigned, const cannot, and var is function-scoped with hoisting behavior.

What's the difference between == and ===?

== performs type coercion, while === compares both value and type

What are primitive data types in JavaScript?

String, Number, Boolean, Null, Undefined, Symbol, and BigInt.

DOM

When should you use querySelector() instead of getElementById()?

Use querySelector() for CSS selectors and more flexible element selection.

What does closest() do?

It finds the nearest ancestor (or itself) matching a CSS selector.

Why is Event Delegation useful?

It lets one event listener handle events for many child elements efficiently.

ES6

How are arrow functions different from regular functions?

They have a shorter syntax and don't have their own this.

Why use template literals?

They make string interpolation and multiline strings easier.

What is destructuring?

It extracts values from arrays or objects into variables.

Async JavaScript

What problem do Promises solve?

They simplify handling asynchronous operations and reduce callback nesting.

Why use async and await?

They make asynchronous code look and behave like synchronous code.

What does the Fetch API return?

It returns a Promise that resolves to a Response object.

Advanced

What is a closure?

A closure allows a function to remember variables from its outer scope.

What is hoisting?

Variable and function declarations are processed before code execution.

What is debouncing?

It delays function execution until the user stops triggering an event.
